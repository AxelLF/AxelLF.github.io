Introduction à CSS

Qu'est ce que CSS ?

CSS pour (Cascading Style Sheets) est un autre langage qui vient compléter le HTML. HTML décrit le contenu sémantique d'un document, CSS va nous servir à gérer le style. Son principe de fonctionnement de base est celui d'imbrication de boîtes auquel on va ajouter des attributs. De base ces attributs peuvent être appliqués sur les balises déjà présentes dans notre document (par exemple <body>). Au besoin on peut ajouter des blocs définis par <div> pour des conteneurs ou par <span> pour des éléments en ligne.

Bien que les attributs puissent être modifiés directement dans une balise il est important de créer et de privilégier leur définition dans une feuille de style CSS externe. Celui-ci peut être lié au fichier HTML via la ligne de commande :

<link rel="stylesheet" type="text/css" href="style.css"/>

Les attributs d'une balise peuvent être modifiés en utilisant des sélecteurs avec la syntaxe suivante:

selecteur { attribut="valeur"}

Positionnement des blocs avec l'attribut "position":

La position d'un élément dans une page peut être définie par l'attribut position.

Cet attribut peut prendre les valeurs suivantes:

static
Comportement par défaut. L'élément est positionné dans le flux.
relative
L'élément est placé par rapport à la position qu'il aurait en static. Il n'est pas supprimé du flux.
absolute
Il est positionné de façon absolue par rapport à la page. Peut avoir des résultats différents selon le navigateur. Il est supprimé du flux on ne conserve pas l'espace normalement occupé par l'élément.
fixed
L'élément ne change pas de position par rapport à la fenêtre. Ceci empêche le défilement lorsque la page est parcourue.
sticky
Définition du MDN : La position de la boîte est calculée en fonction des flux normaux (c'est-à-dire de la position dans le flux normal). Ensuite, la boîte est décalée par rapport à la racine de son flux et par rapport à son bloc contenant.. Dans tous les cas, y compris avec les éléments table, cela n'affecte pas la position des boîtes subséquentes. Lorsqu'une boîte B est positionnée avec sticky, la position de la boîte suivante est calculée comme si B n'était pas décalée. Pour les éléments qui sont rattachés à des tableaux HTML, « position: sticky » aura le même effet que « position: relative ».
