HTML

Structure d'un document html

Structure de base

html fonctionne avec des balises dans des chevrons: <balise>. On distingue deux types de balises: les balises ouvrantes et fermantes par paires et les balises auto-fermantes dites orphelines.

La première ligne donne la version d'html utlisée. HTML5 est obtenu avec <!doctype html>.

Le contenu d'un document html est contenu dans la balise <html>.

Un document html doit contenir une entête <head> contenant les méta-informations du document et un corps <body> pour le contenu.

Le corps du document peut contenir jusqu'à 6 niveaux de titres imbriqués. Ces derniers sont définis par des balises du type <h1> ... </h1>.

La structure de base est donc la suivante: 

 
          
            <!doctype html> 
            <html>
              <head>
                <meta charset="utf-8" />
                <title>  Titre du document </title>  
		  	
                Méta-informations
			
              </head> 
		  
              <body> 
		  
                <h1> Titre de niveau 1 </h1>
		  
                Contenu du niveau 1.
		  
                <h2> Titre de niveau 2 </h2>
                <h3> Titre de niveau 3 </h3>
                <h4> Titre de niveau 4 </h4>
                <h5> Titre de niveau 5 </h5>
                <h6> Titre de niveau 6 </h6>
			
              </body> 
            </html>

Le corps du document peut avoir différents contenus (images, texte,...). Le texte peut être affiché sous forme de paragraphes définis en html par les balises <p> ... <p>. Pour afficher un texte formaté on utilise les balises <pre> ... <pre>, celles-ci peuvent inclure des balises <code> ... <code> pour indiquer que l'on écrit du code informatique.

Les balises <h> et <p> sont des éléments "blocs" (ils s'affichent les uns en dessous des autres, dans le bloc le contenu essaye d'occuper tout l'espace disponible). Les autres éléments sont dis "en ligne" (ils s'affichent les uns à la suite des autres).

Sectionnement du contenu

Le contenu d'une page html peut être réparti dans différentes sections afin d'avoir une structure logique. Il s'agit ici d'une structure sémantique.

Ces sections sont les suivantes :

  • <header> pour du contenu en entête.
  • <nav> pour une section ayant des liens vers d'autres pages ou vers une autre section de cette page.
  • <aside> pour une section dont le contenu est séparé de la structure principale.
  • <article> représente du contenu autonome dans un document, une page, une application, ou un site
  • <section> représente une section générique d'un document. Une section commence généralement avec un titre.
  • <footer> pour le pied de page de la section. 
 
          
            <!doctype html> 
            <html>
              <head>
                <meta charset="utf-8" />
                <title>  Titre du document </title>  		
              </head> 

              <body> 
                <header> 
                </header>

                <nav> 
                </nav>	

                <article> 
                  <section>
                  </section> 
                </article>	  

                <footer> 
                </footer>
              </body> 
            </html>

Outils HTML:

Mise en valeur du texte:

Il est possible de faire ressortir du texte suivant trois degrés d'importance.

Pour mettre un texte un peu en valeur on utilise: <em> ... </em>.

Pour mettre un texte bien en valeur on utilise: <strong> ... </strong>.

Pour faire ressortir visuellement on utilise: <mark> ... </mark>.

Sur la majorité des navigateurs <em> va mettre le texte en italique, <strong> va le mettre en gras et <mark> va le surligner.

Attention: Les balises <br> et <hr> sont des balises héritées des premières versions d'html. Ne pas les utiliser.

Les listes

Il existe trois types de liste, ordonnées, désordonnées et les listes de définition. Celles-ci ont la structure suivante:

Une liste ordonnée (entre balises <ol> ... <ol>):

  1. Element ordonné 1 (chaque élément entre balises <li> ... <li>)
  2. Element ordonné 2
  3. Element ordonné 2

Une liste désordonnée (entre balises <ul> ... <ol>:

  • Element desordonné 1 (chaque élément entre balises <li> ... <li>)
  • Element desordonné 2
  • Element desordonné 3

Une liste de définition (entre balises <dl> ... <dl>:

Le terme défini (chaque élément entre balises <dt> ... <dt>).
La définition(chaque élément entre balises <dd> ... <dd>).

Hyperliens

La balise <a> permet de définir un hyperlien vers une autre page ou un autre endroit de la même page.

La structure d'un hyperlien est la suivante:

< a href="chemin"> Nom du lien <a>

La balise <a> admet différents attributs:

  • "href" indique la cible du lien.
  • "target" indique où le lien doit ouvrir l'affichage et peut avoir des mots-clés.

Mots-clés pour target:

  • "_self" : ouvre le résultat dans la page de navigation déjà ouverte (valeur par défaut. Ex: Page index dans la page courante
  • "_blank" : ouvre le résultat dans un nouvel onglet de navigation. Ex: Page index dans un nouvel onglet
  • "_parent" : ouvre le résultat dans le contexte de navigation HTML5 parent relativement au contexte courant (sans options le résultat est le même que _self). Ex: test parent
  • "_top" : en HTML5 ouvre le résultat dans la page de navigation de plus haut niveau(sans options le résultat est le même que _self). Ex: test top

Pour choisir l'endroit où le lien va afficher le résultat dans un document il faut définir un attribut id. Le nom de cet attribut peut ensuite être utilisé dans le lien avec "#":

< a href="chemin#NomId"> Nom du lien <a>

Attention à ne pas utiliser plusieurs fois le même Id dans une même page.

Les tableaux

En html un tableau est un empilement de lignes comportant des cellules.

Les trois balises à connaitre sont :

  • <table> creation du tableau
  • <tr> ajout d'une ligne
  • <td> ajout d'une cellule

Par exemple:

 
          
            <table>
              <tr>
                <td>A</td>
                <td>B</td>
              </tr>
              <tr>
                <td>C</td>
                <td>D</td>
              </tr>
            </table>

nous donne le tableau suivant:

A B
C D

Caractéristiques d'un tableau:

La balise qui définit l'en-tête d'un tableau est <thead>

La balise qui définit le pied d'un tableau est <tfoot>

La balise qui définit une cellule d'entête ou de pied est <th>, celle-ci peut être utilisée dans les autres lignes du tableau.

Pour fusionner des cellules dans une même ligne on utilise l'attribut "rowspan"

Pour fusionner des cellules sur plusieurs lignes on utilise l'attribut "colspan"

Pour ajouter une légende on utilise <caption>, cet élément doit être placé en première ligne.

Si on reprends le tableau défini ci-dessus celà donne:

 
          
            <table>
              <caption> La légende </caption>
              <thead>
                <th rowspan="2"> </th>
              </thead>
              <foot>
                <th> </th>
              </foot>
              <tr>
                <td>A</td>
                <td>B</td>
              </tr>
              <tr>
                <td>C</td>
                <td>D</td>
              </tr>
            </table>

Pied
A B
