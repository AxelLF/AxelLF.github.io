Introduction à CSS

Qu'est ce que CSS ?

CSS pour (Cascading Style Sheets) est un autre langage qui vient compléter le HTML. Il va nous servir à gérer la forme de notre document. Son principe de fonctionnement de base est celui d'imbrication de boîtes auquel on va ajouter des attributs. De base ces attributs peuvent être appliqués sur les balises déjà présentes dans notre document (par exemple <body>). Au besoin on peut ajouter des blocs définis par <div> pour des conteneurs ou par <span> pour des éléments en ligne.

Bien que les attributs puissent être modifiés directement dans une balise il est important de créer et de privilégier leur définition dans une feuille de style CSS externe. Celui-ci peut être lié au fichier HTML via la ligne de commande :

<link rel="stylesheet" type="text/css" href="style.css"/>

Positionnement des blocs

La position d'un élément dans une page peut être définie par l'attribut position.

Cet attribut peut prendre les valeurs suivantes:

  • static
  • relative
  • absolute
  • fixed
  • sticky
