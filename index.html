Document en html

Structure de base

html fonctionne avec des balises dans des chevrons: <balise> .

La première ligne donne la version d'html utlisée. HTML5 est obtenu avec <!doctype html>.

Le contenu d'un document html est contenu dans la balise <html>.

Un document html doit contenir une entête <head> contenant les méta-informations du document et un corps <body> pour le contenu.

Le corps du document peut contenir jusqu'à 6 niveaux de titres imbriqués. Ces derniers sont définis par des balises du type <h1> ... </h1>.

La structure de base est donc la suivante: 

 
	  
        <!doctype html> 
        <html>
          <head>
            <meta charset="utf-8" />
            <title>  Titre du document </title>  
			
            Méta-informations
			
          </head> 
		  
          <body> 
		  
            <h1> Titre de niveau 1 </h1>
		  
            Contenu du niveau 1.
		  
            <h2> Titre de niveau 2 </h2>
            <h3> Titre de niveau 3 </h3>
            <h4> Titre de niveau 4 </h4>
            <h5> Titre de niveau 5 </h5>
            <h6> Titre de niveau 6 </h6>
			
          </body> 
        </html>

Le corps du document peut avoir différents contenus (images, texte,...). Le texte peut être affiché sous forme de paragraphes définis en html par les balises <p> ... <p>. Pour afficher un texte formaté on utilise les balises <pre> ... <pre>, celles-ci peuvent inclure des balises <code> ... <code> pour indiquer que l'on écrit du code informatique.

Les balises <h> et <p> sont des éléments "blocs" (ils s'affichent les uns en dessous des autres, dans le bloc le contenu essaye d'occuper tout l'espace disponible). Les autres éléments sont dis "en ligne" (ils s'affichent les uns à la suite des autres).

Les balises utiles pour le contenu:

Il est possible de faire ressortir du texte suivant trois degrés d'importance. Sur la majorité des navigateurs ces trois balises ont les effets suivants:

Pour mettre du texte en italique on utilise: <em> ... </em>.

Pour mettre du texte en gras on utilise: <strong> ... </strong>.

Pour mettre du texte en surligné on utilise: <mark> ... </mark>.

Attention: Les balises <br> et <hr> sont des balises héritées des premières versions d'html. Ne pas les utiliser.

Les listes

Il existe trois types de liste, ordonnées, désordonnées et les listes de définition. Celles-ci ont la structure suivante:

Une liste ordonnée (entre balises <ol> ... <ol>):

  1. Element ordonné 1 (chaque élément entre balises <li> ... <li>)
  2. Element ordonné 2
  3. Element ordonné 2

Une liste désordonnée (entre balises <ul> ... <ol>:

  • Element desordonné 1 (chaque élément entre balises <li> ... <li>)
  • Element desordonné 2
  • Element desordonné 3

Une liste de définition (entre balises <dl> ... <dl>:

Le terme défini (chaque élément entre balises <dt> ... <dt>).
La définition(chaque élément entre balises <dd> ... <dd>).
