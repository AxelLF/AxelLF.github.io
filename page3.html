Les formulaires

Il est possible de créer des formulaires en html grâce à la balise <form>. 
Par défault celle-ci doit obligatoirement avoir deux attributs:  action et method.
action : URL pour l'envoi des données.
method : La façon dont les données sont envoyées sur le serveur (entête ou corps). Elle prend la valeur get ou post.

Les champs de saisie sont créés avec la balise <input>, elle peut avoir de nombreux attributs dont la liste complète est disponible ici.

En voici quelque-uns :

  • type: le type de données attendues dans le champ de saisie (ex: text,email...)
  • name: Le nom du contrôle, récupéré par le serveur lors de l'envoi du formulaire. Il donne de quel partie du formulaire il s'agit. Il est indispensable.
  • id: L'identifiant d'un élément, sert de clé, il doit être unique, c'est la valeur de "name".
